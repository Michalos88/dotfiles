# Dotfiles

###copying dotfiles to a new computer

clone this repository
```
git clone https://github.com/bsiddiqui/dotfiles.git ~/dotfiles
```
navigate into dotfiles directory
```
cd ~/dotfiles
```
make the devsetup.sh script executable
```
chmod +x devsetup.sh
```
run the script
```
./devsetup.sh
```
